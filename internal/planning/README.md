# Internal Planning Directory

This directory contains project management materials, planning documents, and internal resources for the Agent-Based Models dissertation research project.

## Contents

- **Research Planning**: Project timelines, milestones, and research objectives
- **Meeting Notes**: Documentation of advisor meetings and research discussions  
- **Literature Reviews**: Notes on relevant papers and theoretical frameworks
- **Methodology Planning**: Experimental designs and statistical analysis plans
- **Progress Tracking**: Research progress reports and status updates

## Files

- `Advanced Critique.txt`: Critical analysis and review materials
- `Templates and initial repo design.txt`: Repository design templates and structure
- `Links and documentation.txt`: Important links and documentation references

## Usage

This directory is for internal project management and is not part of the public research outputs. These materials support the research process but are separate from the publishable code and results.

## Security Note

Ensure no sensitive information (personal data, unpublished results, proprietary methods) is committed to version control in this directory.
